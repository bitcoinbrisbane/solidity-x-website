Standard Solidity Output

contract Token {
    mapping(address => uint256) balances;
    
    function transfer(address to, uint256 amount) 
        external returns (bool success, string memory error) {
        
        uint256 balance = balances[msg.sender];

        if (to == address(0)) {
            balances[msg.sender] -= amount;
            balances[to] += amount;
            return (true, "");
        } else {
            return (false, "Insufficient balance");
        }
    }
}

Getting Started

1

Install Solidity X

Install the transpiler via npm or download the binary directly from our releases page.

npm install -g solidity-x

2

Write Enhanced Solidity

Create your smart contracts using Solidity X's enhanced syntax and features in .solx files.

3

Transpile to Solidity

Run the transpiler to convert your enhanced code to standard Solidity.

solx compile contracts/*.solx --output build/

4

Deploy as Usual

Use the generated Solidity files with Hardhat, Truffle, Remix, or any other Ethereum development framework.
